## üì¶ Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd wiep-website
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Start the development server**
   ```bash
   npm start
   # or
   yarn start
   ```

4. **Open your browser**
   Navigate to `http://localhost:3000` to view the website
